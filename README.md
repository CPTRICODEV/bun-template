## Setup

Make sure to install the dependencies - Server:

```bash
cd server
```

```bash
bun install
```

Make sure to install the dependencies - Front-end:

```bash
cd frontend
```

```bash
bun install
```

## Development

### Server
```bash
cd server
```

```bash
bun run dev
```

### Front-end
```bash
cd frontend
```

```bash
bun run dev
```

## Production

### Front-end

Make sure to build the front-end
```bash
cd frontend
```

```bash
bun run build
```
When the build is done go into the server and just the following commands:

### Server
```bash
cd server
```

```bash
bun run start
```
